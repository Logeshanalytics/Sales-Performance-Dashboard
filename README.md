# Sales-Performance-Dashboard
Excel dashboard for analyzing sales performance by region, product, and salesperson

## Project Overview

This project analyzes sales data and presents insights through an interactive Excel dashboard. It helps stakeholders understand revenue trends, profitability, regional performance, product contribution, and salesperson effectiveness using visual KPIs and charts.

## Objective

* Provide a business-ready dashboard for decision-making
* Track key sales metrics and performance trends
* Identify top-performing regions, products, and salespersons

## Dataset Details

The dataset contains 1,000 sales transactions with the following fields

* Date
* Year
* Region
* Product
* Revenue
* Cost
* Profit
* Profit Margin
* Salesperson

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Excel Formulas
* Slicers
* Conditional Formatting
* Dashboard Design

## Key Metrics

* Total Revenue
* Total Profit
* Profit Margin percentage
* Total Orders
* Monthly Sales Trend
* Region-wise performance
* Product-wise performance
* Salesperson contribution

## Dashboard Features

* KPI cards for revenue, profit, margin, and orders
* Interactive slicers for Year, Region, Product, and Salesperson
* Monthly sales trend line chart
* Regional, product, and salesperson performance charts
* Key insights section for business users

## Key Insights

* West region is the highest revenue contributor
* Laptops generate the highest sales
* Profit margins are consistently above 30 percent
* Sales contribution varies across salespersons

## Business Value

This dashboard helps management quickly identify high-performing regions and products, monitor profitability, and evaluate sales team performance, leading to better strategic decisions.
